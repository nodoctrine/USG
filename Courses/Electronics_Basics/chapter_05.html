5.1 The Ideal Op-Amp

An operational amplifier (op-amp) is a high-gain DC-coupled differential amplifier packaged as a single integrated circuit. It has three signal pins: a non-inverting input (V+), an inverting input (V−), and an output (Vout), plus dual power rails (+Vcc and −Vee) that define its output voltage range.

In circuit analysis we begin with the ideal op-amp model, which makes three assumptions:

  • Open-loop voltage gain AOL is infinite.
  • Input impedance is infinite — no current flows into either input pin.
  • Output impedance is zero — the output behaves as a perfect voltage source.

These assumptions unlock a powerful shortcut. When negative feedback is applied (connecting output back to V−), the op-amp adjusts its output until V+ = V−. This equality is called the virtual short — the inputs are not physically shorted, but feedback forces them to the same potential.

Open-Loop Behaviour

Open-Loop Output:
Vout = AOL × (V+ − V−) AOL (ideal) = ∞

A real LM741 has AOL ≈ 200,000. Even at that finite value, 1 mV of differential input produces 200 V at the output — instantly clipped to the supply rail. Open-loop op-amps are only useful as comparators, not linear amplifiers.

The Virtual Short

When Vout is fed back to V−, any error (V+ − V−) is amplified and fed back to cancel itself. The circuit reaches equilibrium only when V+ = V−.

Two Golden Rules (with negative feedback):
Rule 1: V+ = V− Rule 2: Iin = 0 A

/!\ The inputs are NOT physically wired together. The equality is maintained by the feedback loop, not a short circuit.

5.2 Inverting & Non-Inverting Configurations

Adding two resistors around an op-amp creates the two fundamental closed-loop amplifier configurations. Both are derived using the golden rules from 5.1.

  • The inverting amplifier routes the input through a resistor to V−. The output is phase-reversed (180° flip) relative to the input, and the gain magnitude is set by the resistor ratio.
  • The non-inverting amplifier applies the input directly to V+. The output is in-phase with the input and always has gain ≥ 1.

In both cases, closed-loop gain ACL depends only on the external resistors, not on the op-amp's internal gain. This makes it stable and predictable even as op-amps age or vary between batches.

5.3 Common Op-Amp Circuits

Four standard circuits appear in nearly every analogue design. Each is a direct application of the golden rules.

  • Voltage follower — unity-gain buffer. Isolates a high-impedance source from a low-impedance load without changing the signal.
  • Summing amplifier — adds multiple input voltages, each scaled by its own resistor ratio. Foundation of audio mixers and DACs.
  • Comparator — open-loop op-amp. Output saturates fully high or low depending on which input is larger. Used as a 1-bit decision element.
  • Difference amplifier — amplifies V2 − V1 while rejecting any voltage common to both inputs (common-mode noise). Essential for sensor interfaces in noisy environments.

5.3.1 — How Each Circuit Works ○

Voltage Follower (Unity-Gain Buffer)

Vin ——————————— V+ [Op-Amp] —— Vout —— Load V− ———————————• (direct wire)

Virtual short: V− = V+ = Vin → Vout = Vin. Gain = 1. The value is impedance transformation: the source sees a near-infinite load; the driven circuit sees a near-zero source impedance. No voltage is gained but current drive is massively increased.

Summing Amplifier

V1 ——[R]——• V2 ——[R]——•—— V− —[Rf]—— Vout (feeds back) V3 ——[R]——• V+ —————————————— GND

V− = 0 V (virtual ground). Currents sum independently: Itotal = V1/R + V2/R + V3/R. Vout = −Rf/R × (V1+V2+V3). Each input is isolated from the others — the basis of audio mixing boards.

Comparator

ConditionVoutState
V+ > V−+Vsat (≈ +Vcc)HIGH
V+ < V−−Vsat (≈ −Vee)LOW
V+ = V−UndefinedTransition

Common use: set V− to a fixed reference (e.g. 2.5 V). When the sensor on V+ crosses that threshold the output flips. The output can drive a microcontroller logic input directly.

Difference Amplifier

Output (equal resistors R and Rf):
Vout = (Rf/R) × (V2 − V1)

Common-mode rejection: if 60 Hz mains noise couples equally onto both V1 and V2 wires, the subtraction cancels it completely — only the true signal difference V2−V1 appears at the output. Essential for temperature sensors, load cells, and ECG circuits where signal cables run through noisy environments.

Voltage follower: Vout connects directly to V−. Virtual short forces Vout = Vin with near-infinite input impedance and near-zero output impedance.

5.4 Practical Limitations

Real op-amps deviate from the ideal model in four important ways that must be accounted for in any practical design.

  • Output saturation: Vout is bounded by the supply rails. When the required swing exceeds the supply, the output clips, creating severe distortion.
  • Slew rate (SR): the maximum rate at which Vout can change, in V/µs. Fast, large swings are distorted into ramps rather than the intended waveform.
  • Gain-Bandwidth Product (GBW): ACL × bandwidth = constant. Doubling the gain halves the available bandwidth. This is a fixed property of each op-amp model.
  • Input offset voltage (VOS): a small internal mismatch — typically 1–10 mV — acts as a DC error at the input. Multiplied by ACL, it can corrupt the output in precision circuits.
