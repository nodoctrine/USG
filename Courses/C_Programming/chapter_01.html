1.1 What is a Program?

A program is a sequence of instructions executed one at a time by a computer. Programs are written in a programming language — a formal set of rules for expressing those instructions.

The basic instruction types in any program are:

  • Input — get data from a file, keyboard, network, or sensor
  • Process — perform computations on that data
  • Output — send results to a screen, file, or other device

C is a general-purpose language created in the early 1970s. It compiles directly to machine code, giving programs fast execution and direct hardware access. Most operating system kernels and embedded firmware are written in C.

1.2 Variables & Output

A variable is a named location in memory that stores a value. In C, every variable must be declared with a data type before use. The type tells the compiler how much memory to reserve and how to interpret the stored bits.

Common primitive types:

  • int — whole numbers (e.g., 42, -7)
  • float — single-precision decimal numbers (e.g., 3.14f)
  • double — double-precision decimal numbers
  • char — a single character (e.g., 'A')

Output is produced with printf(), which stands for print formatted. It is declared in the standard header <stdio.h>.

1.3 Your First Program

Every C program must have a main() function. This is the entry point — execution always begins at the first line of main() and ends when main() returns.

The return 0; at the end of main() signals to the operating system that the program finished successfully. A non-zero return value indicates an error.

Figure Figure 1.2 — Program Structure
1 2 3 4 5 6 7 8 9
// Include the standard I/O library #include <stdio.h> // Every C program starts here int main() { printf("My first C program.\n"); printf("I am learning C.\n"); return 0; // 0 = success }
Two printf calls produce two lines of output. The \n is a newline character.
