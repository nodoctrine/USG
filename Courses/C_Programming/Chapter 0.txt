Unions, Bit Manipulation, and Enumerations in C
1. Unions in C

A union is a user-defined data type where all members share the same memory location.
Example
#include <stdio.h> union Data { int i; float f; char str[20]; }; int main() { union Data data; data.i = 10; printf("data.i = %d\n", data.i); data.f = 3.14; printf("data.f = %.2f\n", data.f); printf("data.i after assigning float = %d\n", data.i); return 0; }
Explanation

    All members share the same memory location.
    When one value is assigned, it overwrites the previous value.
    Only one member should be used at a time.

2. Bit Manipulation

Bit manipulation involves working directly with binary bits.
Operators

    & (AND)
    | (OR)
    ^ (XOR)
    ~ (NOT)
    << (Left Shift)
    >> (Right Shift)

Example
#include <stdio.h> int main() { int a = 5, b = 3; printf("a & b = %d\n", a & b); printf("a | b = %d\n", a | b); printf("a ^ b = %d\n", a ^ b); printf("~a = %d\n", ~a); printf("a << 1 = %d\n", a << 1); printf("a >> 1 = %d\n", a >> 1); return 0; }
Explanation

    AND: Both bits must be 1
    OR: At least one bit is 1
    XOR: Bits must be different
    NOT: Flips all bits
    Left Shift: Multiplies by 2
    Right Shift: Divides by 2

3. Enumerations (enum)

An enumeration assigns names to integer constants.
Example
#include <stdio.h> enum Day { SUN, MON, TUE, WED, THU, FRI, SAT }; int main() { enum Day today; today = WED; printf("Value of WED = %d\n", today); return 0; }
Explanation

    By default, values start from 0.
    Each identifier represents an integer value.
    Improves readability and maintainability.





10 real world C programming examples

Below are 10 real C programming examples covering unions, bit manipulation, and enumerations, each with clear, line-by-line explanations:
🧪 1. Union Memory Overwrite Example

#include <stdio.h>

union Data {
    int i;
    float f;
};

int main() {
    union Data d;

    d.i = 10;
    printf("Integer: %d\n", d.i);

    d.f = 5.5;
    printf("Float: %.2f\n", d.f);

    printf("Integer after float assignment: %d\n", d.i);

    return 0;
}

🔍 Explanation

    union Data d; → creates one memory block shared by i and f

    d.i = 10; → stores integer

    d.f = 5.5; → overwrites same memory

    Final print → garbage value because memory now holds float

🧪 2. Union Size Demonstration

#include <stdio.h>

union Test {
    int a;
    double b;
    char c;
};

int main() {
    printf("Size of union: %lu\n", sizeof(union Test));
    return 0;
}

🔍 Explanation

    sizeof(union Test) → equals largest member size

    double is typically 8 bytes → union size = 8

🧪 3. Tagged Union (Safe Usage)

#include <stdio.h>

enum Type { INT, FLOAT };

struct Value {
    enum Type type;
    union {
        int i;
        float f;
    } data;
};

int main() {
    struct Value v;

    v.type = INT;
    v.data.i = 100;

    if (v.type == INT)
        printf("Integer: %d\n", v.data.i);

    return 0;
}

🔍 Explanation

    enum Type → tracks active member

    union inside struct → stores value

    if (v.type == INT) → ensures correct access

🧪 4. Check if Number is Even using Bitwise

#include <stdio.h>

int main() {
    int num = 7;

    if (num & 1)
        printf("Odd\n");
    else
        printf("Even\n");

    return 0;
}

🔍 Explanation

    num & 1 → checks last bit

    If last bit = 1 → odd

    Faster than % 2

🧪 5. Swap Two Numbers using XOR

#include <stdio.h>

int main() {
    int a = 5, b = 10;

    a = a ^ b;
    b = a ^ b;
    a = a ^ b;

    printf("a = %d, b = %d\n", a, b);

    return 0;
}

🔍 Explanation

    XOR cancels bits:

        Step 1: a = a ^ b

        Step 2: b = a ^ b → original a

        Step 3: a = a ^ b → original b

    No temporary variable used

🧪 6. Count Set Bits (1s)

#include <stdio.h>

int main() {
    int num = 13, count = 0;

    while (num) {
        count += num & 1;
        num >>= 1;
    }

    printf("Set bits: %d\n", count);

    return 0;
}

🔍 Explanation

    num & 1 → checks last bit

    num >>= 1 → shift right

    Loop continues until number becomes 0

🧪 7. Set, Clear, Toggle Bit

#include <stdio.h>

int main() {
    int num = 4;

    num |= (1 << 1);  // set bit 1
    num &= ~(1 << 2); // clear bit 2
    num ^= (1 << 0);  // toggle bit 0

    printf("Result: %d\n", num);

    return 0;
}

🔍 Explanation

    (1 << k) → creates bit mask

    |= → set bit

    &= ~ → clear bit

    ^= → toggle bit

🧪 8. Enum with Switch Menu

#include <stdio.h>

enum Menu { ADD = 1, SUB, MUL };

int main() {
    int a = 10, b = 5;
    enum Menu choice = ADD;

    switch(choice) {
        case ADD:
            printf("Sum = %d\n", a + b);
            break;
        case SUB:
            printf("Difference = %d\n", a - b);
            break;
        case MUL:
            printf("Product = %d\n", a * b);
            break;
    }

    return 0;
}

🔍 Explanation

    Enum improves readability

    ADD = 1 → assigns values automatically

    switch → clean control flow

🧪 9. Enum for Days with Loop

#include <stdio.h>

enum Day { SUN, MON, TUE, WED, THU, FRI, SAT };

int main() {
    for (int d = SUN; d <= SAT; d++) {
        printf("%d ", d);
    }

    return 0;
}

🔍 Explanation

    Enum values behave like integers

    Loop iterates from 0 to 6

🧪 10. Bitwise Flags (Real-world Use)

#include <stdio.h>

#define READ  1  // 0001
#define WRITE 2  // 0010
#define EXEC  4  // 0100

int main() {
    int permission = 0;

    permission |= READ;
    permission |= WRITE;

    if (permission & READ)
        printf("Read allowed\n");

    if (permission & WRITE)
        printf("Write allowed\n");

    if (!(permission & EXEC))
        printf("Execute not allowed\n");

    return 0;
}

🔍 Explanation

    Flags stored in bits

    |= → enable permission

    & → check permission

    Efficient alternative to multiple variables

Summary

1. Unions in C (Deep Dive)

A union is a composite data type where all members share the same memory location. This means the size of the union is equal to the size of its largest member.
Memory Model
union Data { int i; // 4 bytes float f; // 4 bytes char str[20]; // 20 bytes };

Total size = 20 bytes (largest member).
Unlike structures, unions overlay memory. Writing to one member destroys the previous content.
Example with Memory Behavior
#include <stdio.h> union Data { int i; float f; }; int main() { union Data d; d.i = 1065353216; printf("As int: %d\n", d.i); printf("Interpreted as float: %f\n", d.f); return 0; }
Explanation

    The integer bit pattern is reinterpreted as a float.
    This demonstrates type punning.
    Useful in low-level programming, but risky if misused.

Advanced Use Case: Tagged Union
#include <stdio.h> enum Type { INT, FLOAT }; struct Value { enum Type type; union { int i; float f; } data; }; int main() { struct Value v; v.type = FLOAT; v.data.f = 3.14; if (v.type == FLOAT) printf("Float: %.2f\n", v.data.f); return 0; }

This avoids undefined behavior by tracking the active member.
2. Bit Manipulation (Rigorous)

Bit manipulation operates at the binary level and is critical in systems programming, embedded systems, and optimization.
Binary Representation
Decimal 5 = 00000101 Decimal 3 = 00000011
Operator Table
Operator 	Name 	Description
& 	AND 	1 if both bits are 1
| 	OR 	1 if at least one bit is 1
^ 	XOR 	1 if bits differ
~ 	NOT 	Inverts bits
<< 	Left Shift 	Shift left (×2)
>> 	Right Shift 	Shift right (÷2)
Example
int a = 5; // 00000101 int result = a << 2;

Result = 20 (00010100)
Bit Masking Techniques
Set a Bit
num = num | (1 << k);
Clear a Bit
num = num & ~(1 << k);
Toggle a Bit
num = num ^ (1 << k);
Check a Bit
if (num & (1 << k)) printf("Bit is set");
Example Program
#include <stdio.h> int main() { int num = 8; // 1000 num |= (1 << 1); // set bit 1 → 1010 num &= ~(1 << 3); // clear bit 3 → 0010 num ^= (1 << 0); // toggle bit 0 → 0011 printf("%d\n", num); return 0; }
Right shift of signed integers is implementation-defined. Be cautious with negative numbers.
3. Enumerations (Advanced)

An enum defines a set of named integer constants.
Default Behavior
enum Day { SUN, MON, TUE };

Values: SUN=0, MON=1, TUE=2
Custom Values
enum ErrorCode { OK = 0, WARNING = 1, ERROR = -1 };
Memory Considerations

    Enums are typically stored as integers.
    Size depends on compiler (usually 4 bytes).

Enum with Switch
#include <stdio.h> enum Day { SUN, MON, TUE }; int main() { enum Day d = MON; switch(d) { case SUN: printf("Sunday"); break; case MON: printf("Monday"); break; case TUE: printf("Tuesday"); break; } return 0; }
Best Practices

    Use enums instead of magic numbers
    Combine with switch for readability
    Use prefixes to avoid naming conflicts

Enums improve maintainability but do not enforce strict type safety in C.
Summary

    Unions: Memory-efficient, but only one active member.
    Bit Manipulation: Powerful for low-level control and optimization.
    Enums: Improve readability using named constants.
