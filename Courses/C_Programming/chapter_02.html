2.1 Conditionals

An if statement lets a program choose what to do based on a condition. If the condition is true, the block runs; otherwise it is skipped.

if (x > 0) {
    printf("positive\n");
} else {
    printf("non-positive\n");
}

The else if clause chains multiple conditions. Only the first true branch executes; if none match, the optional else branch runs.

Lecture 2.1.1 — How an if Statement Executes ○
Step 1: Evaluate the Condition

    ...
}

↑ Is x greater than 0?
Step 2: Branch

x = 5 → condition true → runs the block
x = -3 → condition false → block skipped
Step 3: else Provides the Alternative

if (x > 0) {
    printf("positive");
} else {
    printf("non-positive");
}
The program evaluates the condition inside the parentheses. The result is either true (non-zero) or false (zero).

2.2 Loops

A for loop repeats a block a fixed number of times. It has three parts: an initializer, a condition, and an increment.

for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}

A while loop repeats as long as its condition is true. Use it when the number of iterations is not known in advance.

A do-while loop always executes its body at least once before checking the condition.
